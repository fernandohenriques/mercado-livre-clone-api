### How to run

```
Pelo terminal, entre na pasta onde deseja manter o projeto e execute os seguintes passos:

$ git clone https://github.com/fernandohenriques/mercado-livre-clone-api.git
$ cd mercado-livre-clone-api/
$ cp .env.sample .env
$ npm install
$ npm run dev
```

### To run in Prod

```
$ cd mercado-livre-clone-api/
$ npm install
$ npm run start
```

### Static Analysis

```
$ cd mercado-livre-clone-api/
$ npm run test:lint
```
